# Domains

Mapa de dominios funcionales y tecnicos.

## Dominios troncales ya mapeados

- `transactions`
- `validations`
- `support-and-operations`
- `systems-and-integrations`
- `accounts`
- `users`
- `agreements`

## Como usar esta carpeta

- Usa `transactions` para entender el motor principal del pedido.
- Usa `validations` para entender chequeos, documentos, biometria y bloqueos.
- Usa `support-and-operations` para entender tareas, panel, chat operativo e intervenciones humanas.
- Usa `systems-and-integrations` para entender sistemas, networks, fees, rates e integraciones externas.
- Usa `accounts` para entender cuentas origen/destino, networks, account details y restricciones por contexto.
- Usa `users` para entender identidad, nivel, riesgo, acceso y perfil del usuario dentro de Saldoar.
- Usa `agreements` para entender la ruta de pago con acuerdos propios o de provider, y cómo eso cambia el comportamiento del pedido.
